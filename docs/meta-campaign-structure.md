# Meta Campaign Structure - GitHub Scout Operator

## Campaign Objective

Start with lead generation or sales/conversions once checkout is connected. Until payment is live, optimize for qualified URL scan submissions.

Primary destination:

`netlify-v9-githubscout-ecommerce/operator-shopify-savings.html`

## Core Promise

Paste your Shopify URL. GitHub Scout shows better plugins, embeds, widgets, and third-party software to reduce app spend and improve conversion.

## Campaign 1 - Cost Savings

Audience:

- Shopify store owners
- Ecommerce founders
- DTC operators
- Solo marketers managing app stacks

Creative angles:

- App bill audit
- Plugin spend leak
- App stack bloat
- Cheaper third-party software

Recommended ads:

- `github-scout-operator-shopify-ad-01-paste-url-find-savings.png`

- `github-scout-operator-shopify-ad-04-cut-app-stack-bloat.png`

## Testing Plan

Run three ad sets for seven days:

1. Cost Savings
2. Conversion Lift
3. Operator Plan

Judge by:

- Cost per URL submission
- Landing page form completion rate
- Percentage of leads with app spend above $100/month
- Reply/booking rate after sample report follow-up
- Checkout conversion once payment is connected

## Initial Budget

Small test:

- $20/day per ad set
- 3 ad sets
- 7 days
- Total: $420

Scale only the angle that gets qualified URL submissions, not just cheap clicks.
